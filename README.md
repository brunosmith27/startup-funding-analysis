# Indian Startup Funding Analysis — Business Intelligence from 3,044 Records

Data analysis project extracting business insights from 3,044 Indian 
startup funding records using Python data science tools.

## Project Overview

Built as part of the AI & Machine Learning Graduate Certificate at 
Humber Polytechnic. This project demonstrates the full data analysis 
workflow — from messy raw data through to actionable business insights.

**Tools:** Python · Pandas · Matplotlib · Seaborn

## Problem Statement

The Indian startup ecosystem generates enormous amounts of funding data. 
This project answers 6 key business questions to help investors and 
founders understand where capital flows, which sectors attract funding, 
and what a typical raise looks like.

## Key Business Questions Answered

1. Which cities attract the most startup funding?
2. What is the median startup raise?
3. Which investors are most active?
4. Which industries attract the most capital?
5. How has funding trended over time?
6. What funding types are most common?

## Key Findings

- Bangalore attracted approximately $18B in total funding
- Median startup raise is $1.7M
- Sequoia Capital is the most active investor in the dataset

## Data Cleaning Pipeline

Raw data required significant cleaning including:
- Currency normalisation
- City name standardisation
- Industry category consolidation
- Funding type consolidation
- Handling of missing and malformed values

## Key Skills Demonstrated

- Pandas data cleaning and transformation at scale
- Exploratory data analysis (EDA)
- Data visualisation with Matplotlib and Seaborn
- Translating raw data into business-readable insights
- Dual-chart visualisations for comparative analysis
